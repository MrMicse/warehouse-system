# Warehouse System — Учёт материальных ценностей

Информационная система для автоматизации учёта материальных ценностей предприятия.

## Возможности

- авторизация пользователей;
- разграничение доступа по ролям;
- учёт материалов;
- справочники категорий, складов и поставщиков;
- регистрация прихода и расхода материалов;
- просмотр журнала движений;
- работа с MySQL;
- использование VIEW, хранимых процедур и триггеров.

## Технологии

- C#;
- Windows Forms;
- MySQL / MariaDB;
- XAMPP;
- MySqlConnector;
- GitHub.
